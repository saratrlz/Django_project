### Web app description
Simple webapp of a fruit shop which displays the fruits with their prices

### TOOLS
All useful tools :

    [ ] Run Web app : python3 app/manage.py runserver
    [ ] Create directory : mkdir temp_folder

    
    [ ] Change Directory : cd ./correct_folder
    [ ] Open current directory : explorer .
    [ ] Open VisualStudio code in the current directory : code .

All git tools :

    [ ] Clone projet : git clone https://github.com/saratrlz/Django_project.git>
    [ ] Add all modifications : git add .
    [ ] Attach a message : git commit -m "MyMessage"
    [ ] Push modifications : git push

    [ ] Add local credentials mail : git config user.email <email@mail.com>
    [ ] Add local credentials username : git config user.name <GitUsername>

    [ ] CARE TO CREATE A OAUTH TOKEN : https://github.com/settings/tokens
